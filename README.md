# Taroko Contact List


## Installation

Recommended environment:

1. node `18.17.0`
2. pnpm `8.6.10`

Install dependencies

```sh
pnpm i
```

Create environment variable

```sh
cp .env.example .env
code .env # edit `.env` file and set your `VITE_API_URL`
```

## Development

```sh
pnpm run dev
```

## Build

```sh
pnpm run lint
pnpm run build
```
